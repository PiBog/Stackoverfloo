“Завдання: Написати копію stackoverflow в якому є підтримка: -Реєстрації юзерів -Ролі у юзерів(юзер, адмін, модератор) -Юзери можуть: --Створювати запитання --Редагувати вої вже створені запитанняҐ --Відповідати та коментувати чужі запитання --Реєструватись --Логінитись Реалізовано все має бути на Spring boot2 Захист додатка має бути з використанням OAUTH2. База данних для сайту має бути Postgres Для сервісу потрібно написати хочаб самий простий інтерфейс на HTML/CSS та логіку роботи прописати в JS файлах На HTML/CSS та JS можна приділяти не багато уваги так як це не основне. Основне написати Rest сервіс який приймяє JSON запити та видає JSON відповіді. Переходячи на URL (на локальній машині localhost:port) ми повинні потрапити на головну сторінку на якій можемо зареєструватись або залогінитись, далі коритсувач може переглядати запитання інших користувачів або створити свій. Оцінювати запитання можуть тільки ті в кого є роль модератора(оцінювати можна +1 або -1 до рейтингу як у оригінальному stackoverflow). Користувач який створив запитання має можливіть обрати найкращу відповідь. Адмінка це окрема сторінка на яку можуть логінитись лише адміни, вони можуть перегланути список користувачів та забанити або розбанити користувачів що б ті втратили можливість користуватись сайтом або віднивили відповідно.”
